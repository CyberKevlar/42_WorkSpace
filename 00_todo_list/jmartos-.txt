BADASS

Este proyecto tiene pretende usar GNS3 para simular redes de ordenadores
y dispositivos de red como routers, switches, etc.

La parte 1 del proyecyo conseiste agroso modo en 3 pasos:

1. Instalar GNS3 en docker.
2. Crear una red con 1 host ligero y 1 router FRR.
3. Configurar los routers para que se vean entre ellos y puedan intercambiar informacion.











Vamos a empezar a desglosar el proyecto desde la raiz, fichero a fichero:

                    docker-compose.yml

En este fichero se especifican los servicios que vamos a usar en el proyecto. En este caso vamos a usar GNS3,
un 'programa' que nos permite simular redes de ordenadores y dispositivos de red como routers, switches, etc.

Toda la informacion relacionada con GNS3 y como incubarla en docker la hemos sacado de foros y paginas web
dedicadas a docker en genera y videos de youtube como:

https://voidnull.es/instalacion-y-configuracion-de-gns3/
https://www.youtube.com/watch?v=x9oC1jR8V0I
(PONER TODAS LAS FUENTES DE INFORMACION)

                    DOCKERFILE

En el dockerfile debemos especificar primero que imagen o servicio vamos a usar. En el caso del servicio GNS3
el subject espedifica que tenemos que instalar una imagen con una distro de Linux y todo lo necesario para
que GNS3 funcione correctamente, como las librerias y paquetes con router y dispositivos de red.
(ver dockerfile en ./P1/gns3)
